Shoulder range of motion (ROM) measurement tool

Measure shoulder rotation using Computer Vision.

This website requires access to the user's camera.

User guide

Using computer vision technology, this application is designed for measuring shoulder range of motion and rotation using your camera. This user guide instructs you on how to use this application.
For optimal results, use this tool in a well lit area.

Choose side

To get started, choose below the instructions either the left button to measure the left shoulder, or right button to measure the right shoulder. Your browser will request permission to use your camera, and granting access is necessary for using this application. A video stream will appear shortly displaying the measurements.

Spoken countdown

A spoken countdown from ten (10) will signal the start of the measurement. The results will be read out loud at the end of the countdown.

The measurement process can be restarted by choosing a side again.
